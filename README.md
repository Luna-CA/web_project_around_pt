# Around The U.S.

## Descrição do Projeto

Este é um projeto interativo onde os usuários podem visualizar cartões de lugares ao redor dos EUA, curtir lugares favoritos, editar informações do perfil e adicionar novos cartões.

## Funcionalidades

- Visualizar galeria de cartões com imagens de lugares
- Curtir/descurtir cartões (botão de coração)
- Editar nome e descrição do perfil
- Adicionar novos cartões à galeria
- Excluir cartões existentes
- Visualizar imagens em modal ampliado

## Tecnologias Utilizadas

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (ES6+)
- Metodologia BEM para CSS
- Design responsivo
